git shallow clone

使用--depth参数可以只克隆部分commit历史

git clone –depth=1 https://github.com/MiCode/Xiaomi_Kernel_OpenSource.git

git clone –depth=1 获取完整仓库

git clone –depth=1 https://github.com/MiCode/Xiaomi_Kernel_OpenSource.git
cd Xiaomi_Kernel_OpenSource
git remote set-branches origin ‘*’
git fetch –depth=1 -v
