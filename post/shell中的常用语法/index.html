去除字符串的前缀/后缀

有一个字符串为hello-world, 现在希望去除前缀hell和后缀ld, 得到新的字符串o-wor

1
2
3

string="hello-world"
prefix="hell"
suffix="ld"

方法如下:

1
2
3
4

$ foo=${string#"$prefix"}
$ foo=${foo%"$suffix"}
$ echo "${foo}"
o-wor

判断字符串是否为空

正确做法

#!/bin/sh

STRING=

if [ -z "$STRING" ]; then
    echo "STRING is empty"
fi

if [ -n "$STRING" ]; then
    echo "STRING is not empty"
fi
---------------
输出正确结果：

root@james-desktop:~# ./zerostring.sh
