# sudo: auth account password session
auth       sufficient     pam_tid.so <--- 把这句话加到最上面

关闭"系统偏好设置"右上角的系统更新提示

当存在新的macOS系统更新时，即使“软件更新”中的“检查更新”是关闭状态，只要在点开过“软件更新”这个页面，右上角总会出现恼人的“1”， 提示你更新，我希望消除这个提示。

1
2

defaults write com.apple.systempreferences AttentionPrefBundleIDs 0
killall Dock

之后只要不主动打开“软件更新”这个页面，就不会再有提示，即使不小心点开了，重新运行一遍即可。

https://apple.stackexchange.com/questions/344278/how-can-i-disable-the-red-software-update-notification-bubble-on-the-system-pref
