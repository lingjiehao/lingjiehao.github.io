deb ${HTTP_OR_HTTPS}://${MIRROR_URL}/ubuntu/ ${VERSION_CODENAME} main restricted universe multiverse
deb ${HTTP_OR_HTTPS}://${MIRROR_URL}/ubuntu/ ${VERSION_CODENAME}-updates main restricted universe multiverse

其他

  • USTC配置生成器: https://mirrors.ustc.edu.cn/repogen/
  • 支持https: sudo apt install ca-certificates
  • 替换为https:
    1

    sudo sed -i 's/http/https/g' /etc/apt/sources.list
  • 替换为http:
    1

    sudo sed -i 's/https/http/g' /etc/apt/sources.list
