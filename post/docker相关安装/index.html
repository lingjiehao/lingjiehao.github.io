docker-compose安装

通过apt安装的docker-compose版本太老，所以还是使用官网的方式安装。

  1. 下载

    sudo curl -L "https://github.com/docker/compose/releases/download/1.29.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

  2. 添加可执行权限

    sudo chmod +x /usr/local/bin/docker-compose

  3. 添加bash-completion（可选）

    sudo curl \
    -L https://raw.githubusercontent.com/docker/compose/1.29.1/contrib/completion/bash/docker-compose \
    -o /etc/bash_completion.d/docker-compose

    https://docs.docker.com/compose/install/#install-compose-on-linux-systems

    https://docs.docker.com/compose/completion/#install-command-completion
